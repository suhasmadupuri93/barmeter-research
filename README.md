# barmeter-research

Two things in one repo:

- **`/research`** — research notes on [Barmeter Technologies](https://www.barmetertech.com): what they do, services, careers, tech, business model.
- **`/project`** — a Next.js + Tailwind + TypeScript marketing-site clone inspired by their site, for learning / portfolio purposes. Not affiliated with Barmeter.

## Quick start (project)

```bash
cd project
npm install
npm run dev
```

Open http://localhost:3000.
